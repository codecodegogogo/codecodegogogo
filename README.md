About Me
I build and organize small web projects, static sites, notes, and experiments around practical development workflows.

Right now I am working through the full path from source code to public website: keeping code in GitHub, deploying static projects with Cloudflare Pages, connecting custom domains, and understanding how DNS, nameservers, builds, and hosting fit together.

I care about making things clear enough that I can explain them later. When I learn a tool or fix a deployment problem, I like turning that process into notes, docs, or a cleaner project structure.
